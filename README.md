## Description

This project is a **database class project** developed by **2nd year Software Engineering students at AASTU**. It implements a **complete database system for a smart parking facility** that manages:

* **Users & Security** - Register customers, managers, and admins with secure authentication.
* **Vehicles** - Track license plates and vehicle types registered to each user.
* **Parking Infrastructure** - Manage multiple parking locations and specific slot types (Standard, VIP, Handicap, Electric).
* **Operations** - Track real-time bookings, entry/exit times, and advanced reservations.
* **Financials** - Process payments via Cash, Card, Telebirr, and Mobile Money; manage staff salary payments and penalties.
* **Staff Management** - Manage employee records and shift schedules.
* **Inventory & Analytics** - Monitor revenue, slot usage, and stock availability with automatic alerts.

The system uses **dual databases** to leverage the strengths of both relational and NoSQL paradigms.
