## Description

This project is a **database class project** developed by **2nd year Software Engineering student at Addis Ababa Science and Technology University**.

It implements a **complete database system for a smart parking facility** that manages:

- **Users** - Register customers, managers, and admins (identified by email and phone)
- **Vehicles** - Track vehicles registered to each user
- **Parking Lots** - Manage multiple parking locations
- **Parking Slots** - Different slot types (standard, VIP, handicap, compact, large, electric)
- **Staff** - Manage employees with shift schedules
- **Bookings** - Record parking sessions with start/end times
- **Payments** - Track payments via cash, card, telebirr, and mobile money
- **Reservations** - Allow users to reserve slots in advance
- **Penalties** - Track overstay and other violations
- **Salary Payments** - Manage staff salary records

The system uses **dual databases** to leverage the strengths of both relational and NoSQL paradigms.
